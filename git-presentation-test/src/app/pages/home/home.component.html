Algunos conceptos con Bilbo

fast forward
Sucede cuando en un merge no hay divergencias y git solo tiene que apuntar 'hacia delante' en la rama
Push --force
hace que la rama del repositorio remoto coincida con la local, y se eliminan así todos los cambios en el repositorio remoto que se hayan producido desde que realizaste la última incorporación de cambios
Se comenta que puede ser util con AMEND / REBASE INTERACTIVO ❔
Fork
A fork is a copy of a repository. Forking a repository allows you to freely experiment with changes without affecting the original project.
Submodulos de Git ⁉️
Los submódulos de Git permiten mantener un repositorio de Git como subdirectorio de otro repositorio de Git
. Con submódulos de Git, un repositorio de Git puede incorporar y rastrear el historial de versiones de código externo.
