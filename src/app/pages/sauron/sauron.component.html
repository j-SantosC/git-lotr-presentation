Reset, Revert y Checkout con Sauron

Reset

Para deshacer cambios que no han sido compartidos con nadie más
git reset HEAD~2
  • --soft – The staged snapshot and working directory are not altered in any way.
  • --mixed – The staged snapshot is updated to match the specified commit, but the working directory is not affected. This is the default option. (por defecto)
  • --hard – The staged snapshot and the working directory are both updated to match the specified commit.
Tambien funciona a nivel de archivo! Pude estar interesante 🤔
git reset HEAD~2 foo.ts
